# How to Open Greek Quests Game from Google Drive

## Quick Start Guide

### Option 1: Open Directly from Google Drive (Easiest)

1. **Navigate to Google Drive** in your web browser
2. **Find the folder** "greek game" 
3. **Right-click** on `index.html`
4. **Select** "Open with" â†’ "Google Chrome" (or your preferred browser)
5. The game will load and you can start playing!

### Option 2: Download and Open Locally

1. **Download the entire folder** from Google Drive
2. **Extract** the files if they're in a zip file
3. **Double-click** `index.html` to open in your default browser
4. The game will run locally on your computer

### Option 3: Share with Others

1. **Right-click** on the "greek game" folder in Google Drive
2. **Select** "Share"
3. **Set permissions** to "Anyone with the link can view"
4. **Copy the link** and share it
5. Recipients can download and open `index.html` to play

## Files Included

- `index.html` - Main game file (open this one!)
- `game.js` - Game logic and questions
- `style.css` - Styling and design
- `README.md` - Project documentation

## Requirements

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No internet connection needed after initial load (fonts load from Google)
- All files must be in the same folder

## Troubleshooting

**Game won't load?**
- Make sure all files (index.html, game.js, style.css) are in the same folder
- Try opening index.html directly in a browser instead of through Google Drive preview
- Check browser console for any error messages

**Fonts not loading?**
- You need an internet connection for fonts to load initially
- After first load, fonts are cached
